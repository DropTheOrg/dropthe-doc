| Wikipedia | 34K entidades | Biografias, imagens |
| SEC EDGAR | 6,534 empresas | Receita, lucro liquido |

| AniList | 2,552 titulos | Demograficos, formatos |
| IGDB | 6,838+ jogos | Imagens, plataformas |

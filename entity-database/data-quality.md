# Qualidade de Dados

Manter qualidade na escala de [2 milhoes de entidades](https://dropthe.org) requer pipelines automatizados.

- **Deduplicacao**: 24,492 entidades fundidas, 160,988 campos de dados preservados
- **Slugs**: 155,179 slugs de filmes reconstruidos
- **Limpeza**: 21,839 URLs em campos de data, 2,142 HTML em texto, 1,534 links orfaos corrigidos
- **Pontuacao de completude**: 244,995 entidades pontuadas (0-100)

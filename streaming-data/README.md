# Datos de Streaming

[DropThe](https://dropthe.org) rastrea **700K registros de disponibilidad de streaming** en las principales plataformas, actualizados diariamente.

Para cada pelicula y serie, monitorizamos que plataformas lo tienen, en que regiones, y si esta incluido en suscripcion o requiere alquiler.

## Explorar Contenido de Streaming

- [Peliculas con datos de streaming](https://dropthe.org/movies/) -- 300K+ films
- [Series con datos de streaming](https://dropthe.org/series/) -- 150K+ programas
- [Guias de streaming](https://dropthe.org/guides/) -- Recomendaciones curadas por plataforma

# Arquitetura do Site

[DropThe](https://dropthe.org) e construido como uma rede de dados e midia.

## Estrutura de URLs

| Categoria | Padrao | Exemplo |
|-----------|--------|---------|
| Empresas | `/companies/{slug}/` | [/companies/apple-inc/](https://dropthe.org/companies/apple-inc/) |
| Pessoas | `/people/{slug}/` | [/people/elon-musk/](https://dropthe.org/people/elon-musk/) |
| Filmes | `/movies/{slug}/` | [/movies/the-godfather-1972/](https://dropthe.org/movies/the-godfather-1972/) |
| Series | `/series/{slug}/` | [/series/breaking-bad-2008/](https://dropthe.org/series/breaking-bad-2008/) |
| Jogos | `/games/{slug}/` | [/games/elden-ring/](https://dropthe.org/games/elden-ring/) |
| Criptomoedas | `/crypto/{slug}/` | [/crypto/bitcoin/](https://dropthe.org/crypto/bitcoin/) |

## Tipos de Pagina

- **Paginas de entidades**: 2M+ paginas geradas de dados estruturados
- **Hubs**: [Empresas](https://dropthe.org/companies/), [Pessoas](https://dropthe.org/people/), [Filmes](https://dropthe.org/movies/), [Series](https://dropthe.org/series/), [Jogos](https://dropthe.org/games/), [Criptomoedas](https://dropthe.org/crypto/)
- **Artigos**: [dropthe.org/blog/](https://dropthe.org/blog/)
- **Guias**: [dropthe.org/guides/](https://dropthe.org/guides/)
- **Interativos**: [Neste Dia](https://dropthe.org/on-this-day/), [Comparador](https://dropthe.org/compare/), [Busca](https://dropthe.org/search)

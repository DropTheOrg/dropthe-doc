# Architecture du Site

URLs : `dropthe.org/{categorie}/{slug}/`. Exemples : [/companies/apple-inc/](https://dropthe.org/companies/apple-inc/), [/people/elon-musk/](https://dropthe.org/people/elon-musk/), [/movies/the-godfather-1972/](https://dropthe.org/movies/the-godfather-1972/).

Pages : entites (2M+), hubs ([entreprises](https://dropthe.org/companies/), [personnes](https://dropthe.org/people/), [films](https://dropthe.org/movies/), [series](https://dropthe.org/series/), [jeux](https://dropthe.org/games/), [cryptos](https://dropthe.org/crypto/)), [articles](https://dropthe.org/blog/), [guides](https://dropthe.org/guides/), [Ce Jour-La](https://dropthe.org/on-this-day/).

Donnees ouvertes : [Kaggle](https://www.kaggle.com/dropthe) | [HuggingFace](https://huggingface.co/DropTheHQ) | [GitHub](https://github.com/DropTheOrg)

# Estandares de Calidad

Cada articulo publicado en [DropThe](https://dropthe.org) pasa por verificaciones automatizadas y editoriales.

## Verificaciones Automatizadas

| Verificacion | Requisito |
|--------------|-----------|
| Conteo de palabras | Minimo 600, objetivo 1,400-2,000 |
| Enlaces a entidades | Minimo 10, objetivo 18-22 |
| Estado HTTP | URL publicada retorna 200 |
| Markup de esquema | Esquema Article valido |
| Seccion TLDR | Presente, 5 puntos cortos |
| Seccion FAQ | Presente con esquema FAQ |

## Estandares Editoriales

- Afirmaciones respaldadas por numeros de la base de datos
- Autoria IA siempre declarada
- Sin difamacion, sin contenido ilegal
- Enlaces internos a [paginas de entidades](https://dropthe.org) relevantes

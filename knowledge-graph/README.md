# Graphe de Connaissances

Le [graphe de connaissances DropThe](https://dropthe.org) relie **3.5 millions de relations verifiees** entre 2 millions d'entites.

Chaque page d'entite sur [dropthe.org](https://dropthe.org) est generee a partir du graphe : [entreprises](https://dropthe.org/companies/), [personnes](https://dropthe.org/people/), [films](https://dropthe.org/movies/), [series](https://dropthe.org/series/), [jeux](https://dropthe.org/games/), [cryptos](https://dropthe.org/crypto/).

Sections Gossip, [Ce Jour-La](https://dropthe.org/on-this-day/) et [articles](https://dropthe.org/blog/) sont alimentes par le graphe.

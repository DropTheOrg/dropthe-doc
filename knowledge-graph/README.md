# Grafo de Conocimiento

El [grafo de conocimiento de DropThe](https://dropthe.org) mapea **3.5 millones de relaciones verificadas** entre 2 millones de entidades, creando una de las mayores bases de datos de relaciones independientes en la web.

## Escala

| Metrica | Cantidad |
|---------|----------|
| Relaciones verificadas | 3.5M+ |
| Entidades conectadas | 2M+ |
| Tipos de relacion | 15+ |
| Enlaces transversales | 216,000+ |
| Enlaces familiares | 262,000+ |

## Como Impulsa DropThe

Cada pagina de entidad en [dropthe.org](https://dropthe.org) se genera a partir de consultas al grafo:

- **Entidades relacionadas** muestran personas, empresas y titulos conectados
- **Gossip** muestra conexiones no obvias minadas por el [motor de inteligencia](../intelligence/)
- **Comparar** usa recorrido de grafos para encontrar puntos de comparacion significativos
- **[Un Dia Como Hoy](https://dropthe.org/on-this-day/)** consulta relaciones temporales en todo el grafo

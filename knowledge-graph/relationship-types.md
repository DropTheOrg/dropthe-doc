# Tipos de Relacoes

O [grafo de conhecimento do DropThe](https://dropthe.org) suporta multiplos tipos de relacoes entre categorias.

| Relacao | Exemplo |
|---------|---------|
| trabalha-em | Pessoa → Empresa |
| fundado-por | Empresa → Pessoa |
| publicado-por | Jogo → Empresa |
| desenvolvido-por | Jogo → Empresa |
| produzido-por | Filme → Empresa |
| dirigido-por | Filme → Pessoa |
| atuou-em | Pessoa → Filme/Serie |
| parte-de | Entidade → Franquia |
| subsidiaria-de | Empresa → Empresa |

Uma pessoa pode conectar-se a [empresas](https://dropthe.org/companies/) (emprego), [filmes](https://dropthe.org/movies/) (atuacao), [jogos](https://dropthe.org/games/) (dublagem) e franquias -- tudo visivel na sua [pagina de entidade](https://dropthe.org/people/).

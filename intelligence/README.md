# Motor de Inteligencia

O [motor de inteligencia do DropThe](https://dropthe.org) transforma dados brutos em fatos acionaveis, pontuacoes e classificacoes.

## Componentes

- **[Mineracao de fatos](fact-mining.md)**: 8 mineradores automatizados, 150K+ fatos gerados
- **[DropThe Good Score](good-score.md)**: Pontuacao etica para 100K+ pessoas (0-100)
- **[Inteligencia de nomes](name-intelligence.md)**: 20K+ nomes analisados entre 1.27 milhoes de pessoas
- **[Pontuacao de completude](richness-scoring.md)**: Cada entidade pontuada (0-100) para priorizar enriquecimento

Resultados aparecem nas [paginas de entidades do DropThe](https://dropthe.org), no [Neste Dia](https://dropthe.org/on-this-day/) e nos [artigos](https://dropthe.org/blog/).

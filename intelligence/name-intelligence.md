# Inteligencia de Nombres

[DropThe](https://dropthe.org) mantiene una base de datos de inteligencia de nombres analizando 20K+ nombres unicos de 1.27 millones de entidades de [personas](https://dropthe.org/people/).

Para cada nombre: distribucion de genero, distribucion de nacionalidad, tendencias por decada, distribucion zodiacal y las 10 personas mas notables.

# Puntuacion de Completitud

Cada entidad en la [base de datos DropThe](https://dropthe.org) recibe una puntuacion de completitud de datos de 0 a 100.

Las puntuaciones impulsan la prioridad de enriquecimiento -- entidades con baja puntuacion y alta popularidad se enriquecen primero, asegurando que las paginas mas visitadas en [dropthe.org](https://dropthe.org) tengan los mejores datos.

# Documentacao do DropThe

Documentacao tecnica do [DropThe](https://dropthe.org) -- uma rede independente de dados e midia.

## O que e o DropThe?

[DropThe](https://dropthe.org) e uma das maiores [bases de dados de entidades](https://dropthe.org/companies/) independentes na web, com mais de **2 milhoes de entidades** conectadas por **3.5 milhoes de relacoes** em entretenimento, tecnologia, financas, gaming e mais.

Cada entidade -- seja uma [empresa](https://dropthe.org/companies/), uma [pessoa](https://dropthe.org/people/), um [filme](https://dropthe.org/movies/) ou uma [criptomoeda](https://dropthe.org/crypto/) -- tem uma pagina dedicada com dados estruturados e inteligencia especifica.

## Sistemas Principais

- **[Base de dados de entidades](entity-database/)** -- 2M+ entidades em 17 categorias
- **[Grafo de conhecimento](knowledge-graph/)** -- 3.5M relacoes verificadas
- **[Dados de streaming](streaming-data/)** -- 700K registros de disponibilidade
- **[Motor de inteligencia](intelligence/)** -- Mineracao de dados, pontuacoes e classificacoes eticas
- **[Pipeline de conteudo](pipeline/)** -- Sistema de producao de artigos baseados em dados
- **[Pesquisa e artigos](research/)** -- Analises publicadas com dados de entidades

## Principios

- **Independente**: Sem patrocinadores corporativos, sem vies editorial
- **Verificado**: Dados cruzados entre multiplas fontes
- **Gratuito**: Todas as paginas sao acessiveis na [plataforma DropThe](https://dropthe.org)
- **Transparencia IA**: Conteudo com assistencia de IA sempre declarado

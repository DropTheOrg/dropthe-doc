# Documentacion de DropThe

Documentacion tecnica de [DropThe](https://dropthe.org) -- una red de datos y medios independiente.

## Que es DropThe?

[DropThe](https://dropthe.org) es una de las mayores [bases de datos de entidades](https://dropthe.org/companies/) independientes en la web, con mas de **2 millones de entidades** conectadas por **3.5 millones de relaciones** en entretenimiento, tecnologia, finanzas, gaming y mas.

Cada entidad -- ya sea una [empresa](https://dropthe.org/companies/), una [persona](https://dropthe.org/people/), una [pelicula](https://dropthe.org/movies/) o una [criptomoneda](https://dropthe.org/crypto/) -- tiene una pagina dedicada con datos estructurados, entidades relacionadas e inteligencia especifica.

## Sistemas Principales

- **[Base de datos de entidades](entity-database/)** -- 2M+ entidades en 17 categorias
- **[Grafo de conocimiento](knowledge-graph/)** -- 3.5M relaciones verificadas
- **[Datos de streaming](streaming-data/)** -- 700K registros de disponibilidad
- **[Motor de inteligencia](intelligence/)** -- Mineria de datos, puntuaciones y clasificaciones eticas
- **[Pipeline de contenido](pipeline/)** -- Sistema de produccion de articulos basados en datos
- **[Investigacion y articulos](research/)** -- Analisis publicados con datos de entidades

## Principios

- **Independiente**: Sin patrocinadores corporativos, sin sesgo editorial
- **Verificado**: Datos cruzados entre multiples fuentes
- **Gratuito**: Todas las paginas de entidades son accesibles en la [plataforma DropThe](https://dropthe.org)
- **Transparencia IA**: El contenido que usa asistencia de IA siempre lo declara
